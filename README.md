# 🌐 Horizen

Horizen é um projeto de e-commerce moderno desenvolvido com **Angular** e **Tailwind CSS**, focado inicialmente em uma experiência **mobile-first**, garantindo usabilidade e performance em dispositivos móveis antes de expandir para o ambiente desktop.

## 🚀 Objetivo

Criar uma plataforma de e-commerce rápida, responsiva e escalável, com uma experiência de usuário fluida e design moderno.

## 🧱 Tecnologias Utilizadas

- **Angular** — Framework frontend para construção da aplicação
- **Tailwind CSS** — Estilização rápida e responsiva
- **TypeScript** — Tipagem estática para maior confiabilidade
- **HTML5 & CSS3**

## 📱 Abordagem Mobile-First

O projeto segue o conceito **mobile-first**, o que significa:

- Interface projetada primeiramente para telas pequenas
- Melhor performance em dispositivos móveis
- Expansão progressiva para tablets e desktops
- Layout adaptável com breakpoints bem definidos

## 🎨 Design e UI

- Interface limpa e moderna
- Componentes reutilizáveis
- Responsividade com Tailwind
- Foco em UX (User Experience)

## 📦 Funcionalidades (Planejadas)

- 🛍️ Catálogo de produtos
- 🔎 Busca e filtros
- 🛒 Carrinho de compras
- ❤️ Lista de favoritos
