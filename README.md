# 🌐 Horizen

Horizen é um projeto de e-commerce moderno inspirado fortemente no design do sita da Amazon, foi desenvolvido com **Angular** e **Tailwind CSS** priorizando o **mobile-first**, garantindo usabilidade e performance em dispositivos móveis antes de expandir para o ambiente desktop.

## 🚀 Objetivo

Criar uma plataforma de e-commerce completa, responsiva e escalável, com uma experiência de usuário agradável e design moderno.

## 🧱 Tecnologias Utilizadas

- **Angular** — Framework frontend para construção da aplicação
- **Tailwind CSS** — Estilização rápida e responsiva
- **TypeScript** — Tipagem estática para maior confiabilidade
- **HTML5 & CSS3**

## 📱 Abordagem Mobile-First

O projeto segue o conceito **mobile-first**, o que significa:

- Interface projetada primeiramente para telas pequenas
- Melhor performance em dispositivos móveis
- Expansão progressiva para tablets e desktops
- Layout adaptável com breakpoints definidos

## 🎨 Design e UI

- Interface limpa e moderna
- Componentes reutilizáveis
- Responsividade com Tailwind

## 📦 Funcionalidades

- 🛍️ Catálogo de produtos
- 🔎 Busca e filtros
- 🛒 Carrinho de compras com persistência
- 💵 Página de Checktout e Confirmação
